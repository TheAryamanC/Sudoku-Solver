# Sudoku Solver

A simple Python Sudoku solver (with text based & image-based puzzle input).

## Installation

1. Create or activate a virtual environment:

```bash
python3 -m venv .venv
source .venv/bin/activate
```

2. Install dependencies:

```bash
pip install -r requirements.txt
```

## Running the solver

From the repository root:

```bash
python3 main.py
```

Then choose one of the menu options:

1. Manual input
2. Import from image
3. Use sample puzzle
4. Exit
